# 🚀 데이터 수집 빠른 시작 가이드

**5분만에 첫 데이터 수집 완료하기!**

---

## 1️⃣ 예측 실행하기 (1분)

### 웹 UI 사용
```
1. 브라우저에서 http://localhost:8000 접속
2. "간단 예측" 탭 클릭
3. 예시 입력:
   - 감정가: 300000000
   - 물건종류: 아파트
   - 지역: 서울
   - 면적: 85
   - 경매회차: 1
4. "AI 예측하기" 클릭
5. ✅ 자동으로 DB 저장됨!
```

### 또는 명령줄 사용
```bash
curl "http://localhost:8000/predict?start_price=300000000&property_type=아파트&region=서울&area=85"
```

## 2️⃣ 미검증 예측 확인 (30초)

```bash
python verify_results.py list
```

**출력 예시:**
```
📋 검증 대기 중인 예측 (1건):
----------------------------------------------------------------------------------------------------
No   사건번호                 감정가             예측 낙찰가        예측일시
----------------------------------------------------------------------------------------------------
1    PREDICT-1770519225    300,000,000원   108,147,565원  2026-02-08 11:53:45
----------------------------------------------------------------------------------------------------
```

**👉 사건번호를 복사하세요: `PREDICT-1770519225`**

---

## 3️⃣ 실제 낙찰가 입력 (2분)

### 방법 A: CSV 파일 (추천)

**1. CSV 파일 작성 (results.csv)**
```csv
사건번호,실제낙찰가,낙찰일자
PREDICT-1770519225,280000000,2024-03-20
```

**2. 업로드**
```bash
python verify_results.py csv results.csv
```

### 방법 B: 명령줄 직접 입력
```bash
python verify_results.py verify "PREDICT-1770519225" 280000000 "2024-03-20"
```
