# 🌐 웹 검증 기능 사용 가이드

웹 브라우저에서 직접 예측 검증하고 CSV로 내보내기!

작성일: 2026-02-09

---

## ✨ 새로운 기능

### 1. 웹에서 직접 검증하기
- 검증 대기 중인 예측에 실제 낙찰가를 브라우저에서 바로 입력
- 클릭 한 번으로 검증 완료
- 실시간으로 정확도 업데이트

### 2. CSV 다운로드
- 미검증 예측 목록을 CSV 파일로 다운로드
- Excel에서 편집 후 다시 업로드 가능
- 대량 작업에 최적화

---

## 📍 사용 방법

### 웹 브라우저 검증 (추천)

**1단계: 정확도 탭 열기**
```
http://localhost:8000 → 📊 정확도 탭 클릭
```

**2단계: 검증 대기 목록 확인**
- "⏳ 검증 대기 중" 섹션에서 미검증 예측 확인
- 각 예측마다 입력 폼이 표시됩니다

**3단계: 실제 낙찰가 입력**
```
┌─────────────────────────────────────────────┐
│ PREDICT-1770519225                          │
│ 감정가: 300,000,000원 | 예측: 108,147,565원 │
│                                             │
│ 실제 낙찰가 (원)    낙찰일자                │
│ [245000000____] [2024-03-20] [✅ 검증]     │
└─────────────────────────────────────────────┘
```

**4단계: 검증 클릭**
- "✅ 검증" 버튼 클릭
- 성공 메시지 확인
- 2초 후 자동으로 정확도 업데이트

---

### CSV 다운로드 방법

**1단계: CSV 다운로드**
```
정확도 탭 → "⏳ 검증 대기 중" 섹션 → "📥 CSV 다운로드" 버튼 클릭
```

**2단계: 다운로드 파일 확인**
파일명 형식: `unverified_predictions_YYYY-MM-DD.csv`

**파일 내용 예시:**
```csv
사건번호,실제낙찰가,낙찰일자
PREDICT-1770635366,,
PREDICT-1770519225,,
```

**3단계: Excel에서 편집**
```csv
사건번호,실제낙찰가,낙찰일자
PREDICT-1770635366,320000000,2024-03-25
PREDICT-1770519225,280000000,2024-03-26
```

**4단계: CSV 업로드**
```bash
python verify_results.py csv unverified_predictions_2026-02-09.csv
```

## 🔄 워크플로우 비교

### 방법 A: 웹 검증 (빠른 소량)
```
1. 브라우저에서 정확도 탭 열기
2. 실제 낙찰가 직접 입력
3. 검증 버튼 클릭
4. 즉시 완료!

⏱️ 시간: 건당 10초
✅ 장점: 즉시 확인, 오류 피드백
📊 최적: 1-5건 검증
```

### 방법 B: CSV 다운로드 + 업로드 (대량)
```
1. CSV 다운로드
2. Excel에서 일괄 편집
3. CSV 업로드
4. 완료!

⏱️ 시간: 10-50건을 10분
✅ 장점: 대량 처리, 오프라인 작업
📊 최적: 10건 이상 검증
```

## 📊 검증 후 확인 사항

### 웹 대시보드
```
http://localhost:8000 → 📊 정확도 탭
```

**확인 가능한 정보:**
- ✅ 총 예측 건수
- ✅ 검증 완료 건수
- ✅ 평균 오차율
- ✅ 평균 오차 금액
- ✅ 최고/최저 정확도
- ✅ 검증된 예측 상세 목록

### 명령줄 리포트
```bash

```

## 🎯 권장 사용 패턴

### 매일 루틴 (웹 검증)
```
1. 아침: 법원 경매 사이트에서 낙찰가 확인
2. 바로 웹 브라우저에서 입력 (1-3건)

```

```
일요일: 정확도 분석
```

## 📝 예시

### 예시 1: 웹에서 1건 검증

**상황:** 오늘 낙찰 완료된 물건 1건

```
1. http://localhost:8000 접속
2. 📊 정확도 탭 클릭
3. 검증 대기 목록에서 해당 사건번호 찾기
4. 실제 낙찰가 입력: 245000000
5. 낙찰일자 선택: 2024-03-20
6. ✅ 검증 클릭
7. 성공 메시지 확인
8. 정확도 자동 업데이트 확인
```

### 예시 2: CSV로 10건 일괄 검증

**상황:** 이번 주 낙찰 완료 10건

```
1. 📊 정확도 탭 → 📥 CSV 다운로드
2. Excel 열기
3. 10건의 낙찰가 일괄 입력
   PREDICT-001,245000000,2024-03-20
   PREDICT-002,180000000,2024-03-21
   ...
4. CSV 저장 (UTF-8)
5. python verify_results.py csv 파일명.csv
6. 웹에서 정확도 확인
```
